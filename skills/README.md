# Skill 广场 · 精选索引

这里收录课程同学推荐、经维护者挑选的 **AI Agent Skills**（如 Cursor / Claude Code 的 `SKILL.md` 与可复用工作流）。日常分享与讨论请发到 Discussions 的 [Skill 广场](https://github.com/DaSE-VibeCoding/SummerCamp-2026/discussions/categories/skill-square)。

## 如何投稿

1. 先搜索 [Skill 广场](https://github.com/DaSE-VibeCoding/SummerCamp-2026/discussions/categories/skill-square) 是否已有相似推荐。
2. 使用「分享 Skill」讨论模板发帖，标题形如 `【Skill】短名称`。
3. 写清真实使用感受、获取方式与安装步骤；完整文件优先外链，勿粘贴密钥或私有配置。

安装失败、报错排查请走 [课程答疑](https://github.com/DaSE-VibeCoding/SummerCamp-2026/issues/new?template=course-q-and-a.yml)，广场帖专注推荐与交流。

## 如何入选本索引

维护者会不定期从 Discussion 中挑选条目写入下表。入选标准：

- 有真实使用说明，不是纯链接搬运
- 公开链接可访问，内容无敏感信息
- 至少有 1 条同学反馈，或维护者试用通过

过时或失效的帖子可将标题改为 `【已归档】`；本表对应行也会同步标注或移除。

## 维护者设置（首次上线必做）

模板文件无法自动创建 Discussions 分类。仓库管理员需在 GitHub 上完成一次：

1. 打开仓库 **Settings → General → Features → Discussions**（确保已开启）。
2. 进入 **Discussions → 分类设置**（或 Categories），新建分类：
   - 显示名：`Skill 广场`
   - slug / 标识：`skill-square`
3. 确认仓库默认分支上已有 `.github/DISCUSSION_TEMPLATE/skill-square.yml`（文件名须与分类 slug 一致，GitHub 会自动关联表单）。

完成后，同学可通过 [新建讨论 · Skill 广场](https://github.com/DaSE-VibeCoding/SummerCamp-2026/discussions/new?category=skill-square) 投稿。
