## 🎯 项目简介

SummerCamp-2026 是面向开发者的夏季编程训练营，专注于 **Vibe Coding**（氛围编程）理念。通过沉浸式的项目实践，帮助参与者掌握现代开发工具与协作流程。

### 什么是 Vibe Coding？

Vibe Coding 强调：
- 🎨 **直觉驱动**：通过自然语言描述需求，让 AI 辅助生成代码
- 🚀 **快速迭代**：专注于产品原型，减少重复性编码工作
- 🤝 **协作优先**：利用现代工具链实现高效团队协作
- 📚 **学习即创造**：在实战中掌握新技术

## ✨ 课程特色

| 特色 | 说明 |
|------|------|
| **AI 辅助开发** | 使用 Claude、GPT 等 AI 工具加速开发流程 |
| **实战项目** | 从 0 到 1 完成完整项目开发 |
| **代码审查** | 专业导师团队提供 Code Review 指导 |
| **社区驱动** | 建立长期技术交流社区 |
| **成果展示** | 优秀项目将获得推广资源支持 |

## 📚 学习资源

### 推荐工具

- **IDE**: VS Code / Cursor / Trae
- **AI 助手**: Claude Code / GitHub Copilot / ChatGPT
- **版本控制**: Git + GitHub
- **协作平台**: GitHub Issues / Discussions

### 学习路径

1. **基础阶段**：熟悉开发环境与 AI 工具使用
2. **实践阶段**：完成指定项目任务
3. **优化阶段**：代码重构与性能优化
4. **展示阶段**：项目演示与经验分享

## 🤝 参与贡献

我们欢迎所有形式的贡献！

### 如何参与？

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

### 贡献指南

- 确保代码符合项目规范
