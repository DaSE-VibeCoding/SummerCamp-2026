# SummerCamp-2026 社区准则

## 我们希望的社区

这里是互相学习、提出问题和寻找伙伴的课程空间。请尊重不同基础，帮助对方理解思路，而不是替对方完成课程作业或提交作品。

## 答疑怎么闭环

1. 先搜索相似 Issue，再通过“课程答疑”表单说明问题目标和已尝试步骤。
2. 愿意协助的同学评论“我来看看”；维护者将状态改为 `认领中`。
3. 提问者确认已解决后，维护者添加 `已解决` 并关闭 Issue。
4. 可复用的答案会整理为 FAQ。

## 怎样找队友

在 Discussions 的“寻找队友”发布你的兴趣、已有技能、希望寻找的技能和预计投入；在“项目招募”说明项目目标和剩余名额。组队成功后把标题改为 `【已组队】`，招满后改为 `【已满】`。

## 怎样分享 Skill

Skill 广场用于推荐 **AI Agent Skills**（如 Cursor / Claude Code 的 `SKILL.md` 与可复用工作流），不是个人技术能力名片，也不是装机答疑区。

1. 先搜索 [Skill 广场](https://github.com/DaSE-VibeCoding/SummerCamp-2026/discussions/categories/skill-square) 是否已有相似推荐。
2. 使用「分享 Skill」表单发帖，标题形如 `【Skill】短名称`；写清真实使用感受、获取方式与安装步骤。
3. 完整 Skill 文件优先用公开仓库或 gist 外链；可在正文放精简摘要。安装失败、报错请走「课程答疑」Issue。
4. 优质推荐可能被维护者写入 [精选索引](../skills/README.md)。过时或失效可将标题改为 `【已归档】`。

分享时请额外遵守：

- 不粘贴密钥、私有端点、含个人路径的配置或未脱敏日志。
- 不完整粘贴他人未授权的商业 Skill；注明来源与许可证。
- 只写用途与公开链接，不发布可执行攻击或利用类内容。

## 安全与尊重

- 不发布密码、Token、Cookie、学号、电话、住址、私人聊天记录或他人的个人信息。
- 不进行歧视、骚扰、人身攻击、恶意刷屏或未经同意的商业推广。
- 不确定内容是否适合公开时，先询问课程维护者。
- 发现不当内容时，请 @ 课程维护者；维护者可编辑、锁定、转移或删除相关内容。
